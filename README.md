# 2025 동서대학교 졸업식 랜딩페이지

모바일 전용 졸업식 포토부스 안내 페이지입니다.

## 🚀 시작하기

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 열기

### 3. 프로덕션 빌드
```bash
npm run build
npm start
```

## 🎨 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Font**: Pretendard (고딕체)
- **Deploy**: Vercel
